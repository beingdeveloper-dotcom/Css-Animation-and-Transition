Transition Vs Animation

Transition:

Transitions are when the browser animates from one state to another (A to B). Theyâ€™re usually triggered by an action such hovering over an element, or adding or removing a class using JavaScript. Transitions enable you to define the transition between two states of an element. Different states may be defined using pseudo-classes like :hover or :active or dynamically set using JavaScript.

Animation:

Animations are more involved, and let you create sequences of animations with as many keyframes as you need along the way. They trigger automatically, and can loop.
